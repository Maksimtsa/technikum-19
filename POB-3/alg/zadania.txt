ZADANIE 1 – System ocen uczniów

Utwórz klasę `Uczen`, a następnie wykonaj operacje algorytmiczne na tablicy obiektów.

Utwórz klasę:
class Uczen
{
    public string Imie { get; set; }
    public double Srednia { get; set; }

    public Uczen(string imie, double srednia)
    {
        Imie = imie;
        Srednia = srednia;
    }
}
W programie głównym:
Utwórz tablicę 5 uczniów.
Wykonaj algorytmy:

1.  Wyszukaj ucznia o podanym imieniu (wyszukiwanie liniowe).
2.  Znajdź ucznia z najwyższą średnią.
3.  Znajdź ucznia z najniższą średnią.
4.  Oblicz średnią ocen całej klasy.
5.  Posortuj uczniów rosnąco według średniej (np. sortowanie bąbelkowe).

ZADANIE 4 – Elektroniczny dziennik

Klasa z metodą:

class Uczen
{
    public string Imie { get; set; }
    private int[] Oceny;

    public Uczen(string imie, int[] oceny)
    {
        Imie = imie;
        Oceny = oceny;
    }

    public double ObliczSrednia()
    {
        int suma = 0;
        for (int i = 0; i < Oceny.Length; i++)
        {
            suma += Oceny[i];
        }
        return (double)suma / Oceny.Length;
    }
}

Zadania
1. Utwórz tablicę uczniów.
2. Wyszukaj ucznia o najwyższej średniej.
3. Zlicz uczniów ze średnią powyżej 4.0.
4. Posortuj uczniów malejąco według średniej.
5. Wyszukaj ucznia po imieniu (wyszukiwanie binarne – po wcześniejszym sortowaniu alfabetycznym).

ZADANIE 5 – Ranking graczy

Klasa:

class Gracz
{
    public string Nick { get; set; }
    public int Punkty { get; set; }

    public Gracz(string nick, int punkty)
    {
        Nick = nick;
        Punkty = punkty;
    }
}


Wymagania
1. Znajdź najlepszego gracza.
2. Znajdź najgorszego gracza.
3. Policz średnią liczbę punktów.
4. Posortuj ranking malejąco.
5. Wyszukaj gracza po nicku.
6. Usuń gracza z rankingu.
7. Skopiuj ranking do nowej tablicy.
